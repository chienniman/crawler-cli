# web-crawer-selenium-cli
網頁自動爬蟲工具

## 環境
* Python 3.11.4
* Chromedriver 114.0.5735.110

## 安裝
* 驅動須對應瀏覽器版本[chromedriver](https://chromedriver.chromium.org/downloads)

### 取得

```bash
git clone https://github.com/chienniman/web-crawer-selenium.git
```

### 根目錄安裝

```bash
pip install selenium
```

### 運行功能
`cd`進入資料夾後，運行

```bash
python start.py  
```
